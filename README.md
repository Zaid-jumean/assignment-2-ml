# Assignment 2 - Machine Learning Foundations

This project predicts bike rental demand using supervised machine learning models.

## Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Main Tasks
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Hyperparameter Tuning
- Model Evaluation and Comparison

## Evaluation Metrics
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
- R² Score

## Final Model
The tuned Random Forest model produced the best overall results and was selected as the final model.

## Dataset
Bike Sharing Dataset (`hour.csv`)
